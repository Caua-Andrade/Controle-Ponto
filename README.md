## 📌 Sobre o projeto

O **Controle de Ponto** é uma aplicação full stack desenvolvida com foco em praticar conceitos reais de desenvolvimento backend e frontend.

O sistema foi pensado especificamente para **estagiários**, que normalmente possuem uma jornada reduzida de **6 horas por dia**, trazendo uma regra de negócio simples, mas realista.

A aplicação permite registrar horários de entrada e saída, além de organizar essas informações de forma clara para o usuário.
